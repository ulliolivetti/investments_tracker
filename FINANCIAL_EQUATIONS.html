1. Average Purchase Price (Preço Médio)

What it measures

When you buy shares of the same company multiple times at different prices, this tells you the true average price you paid per share across all purchases. Think of it like averaging the unit price of materials bought in multiple batches on a construction project.

Formula

\[ \bar{P} = \frac{\sum_{i=1}^{n} (Q_i \cdot P_i)}{\sum_{i=1}^{n} Q_i} \]
  • \(Q_i\) = quantity bought in transaction \(i\)
  • \(P_i\) = price paid per share in transaction \(i\)
  • \(n\) = total number of buy transactions for this stock

Excel → Python

def average_price(transactions: list[dict]) -> float:
    buys = [t for t in transactions if t["action"] == "Compra"]
    total_cost = sum(t["quantity"] * t["price"] for t in buys)
    total_quantity = sum(t["quantity"] for t in buys)
    return total_cost / total_quantity if total_quantity > 0 else 0

Concrete example — Coface (EPA:COFA)

✓ Excel: Preço médio = 15.02210526
Why it matters: Your average price is your personal break-even point. If the stock trades above this, you're in profit. Below, you're at a loss — even if the stock is objectively "up" since you started investing.

2. Invested Value (Valor Comprado)

What it measures

The total amount of real money you put into a stock. This is your cost basis — what you actually spent.

Formula

\[ V_{\mathrm{invested}} = Q_{\mathrm{total}} \times \bar{P} \]
  • \(Q_{\mathrm{total}}\) = total shares currently held
  • \(\bar{P}\) = average purchase price

Excel → Python

def invested_value(quantity: float, avg_price: float) -> float:
    return quantity * avg_price

Concrete example — Coface

\[ V_{\mathrm{invested}} = 19 \times 15.02 = \textbf{285.42} \text{ EUR} \]

3. Current Value (Valor Atual)

What it measures

What your position is worth right now at market prices. This changes every second the market is open.

Formula

\[ V_{\mathrm{current}} = Q_{\mathrm{total}} \times P_{\mathrm{current}} \]

Excel → Python

def current_value(quantity: float, current_price: float) -> float:
    return quantity * current_price

Concrete example — Coface

\[ V_{\mathrm{current}} = 19 \times 16.45 = \textbf{312.55} \text{ EUR} \]

4. Unrealized Gain/Loss (Valorização €)

What it measures

The profit or loss on a position you still hold. It's called "unrealized" because it's not real money yet — you only realize it when you sell. Like owning land that has appreciated in value: you're richer on paper, but you only get the cash when you sell the land.

Formula

\[ G_{\mathrm{unrealized}} = V_{\mathrm{current}} - V_{\mathrm{invested}} \]

Excel → Python

def unrealized_gain(current_value: float, invested_value: float) -> float:
    return current_value - invested_value

Concrete examples

Coface (gain):

\[ G_{\mathrm{unrealized}} = 312.55 - 285.42 = \textbf{+27.13} \text{ EUR} \]

TFI — loss example:

\[ G_{\mathrm{unrealized}} = 102.00 - 107.53 = \textbf{-5.53} \text{ EUR} \]

5. Unrealized Gain Percentage (Valorização %)

What it measures

The same gain expressed as a percentage of your original investment. This lets you compare performance across positions of different sizes — a €27 gain on a €285 investment is much more impressive than a €27 gain on a €2,850 investment.

Formula

\[ G_{\%} = \frac{V_{\mathrm{current}} - V_{\mathrm{invested}}}{V_{\mathrm{invested}}} \times 100 \]

Excel → Python

def unrealized_gain_percent(current_value: float, invested_value: float) -> float:
    if invested_value == 0:
        return 0
    return (current_value - invested_value) / invested_value * 100

Concrete example — Coface

\[ G_{\%} = \frac{312.55 - 285.42}{285.42} \times 100 = \frac{27.13}{285.42} \times 100 = \textbf{+9.50\%} \]
✓ Excel: Valorização % = 0.09505290449 (= 9.50%)

6. Portfolio Weight (Peso da Ação)

What it measures

What percentage of your total portfolio is concentrated in one stock. Civil engineering analogy: load distribution across columns — if one column carries 50% of the load, a failure there brings down the whole structure. Diversification means spreading the weight.

Formula

\[ W_i = \frac{V_{\mathrm{current},\,i}}{\displaystyle\sum_{j=1}^{n} V_{\mathrm{current},\,j}} \]

Where \(i\) is one specific stock and the denominator is the sum of all current values across all \(n\) stocks.

Excel → Python

def portfolio_weights(holdings: list[dict]) -> dict:
    total = sum(h["current_value"] for h in holdings)
    return {
        h["ticker"]: h["current_value"] / total
        for h in holdings
        if total > 0
    }

7. Dividend Yield (Rendimento dos Dividendos)

What it measures

The annual income you receive from a stock expressed as a percentage of its current price. If a stock pays €2 per share and costs €40, you get 5% of your investment back in cash every year — regardless of whether the stock price goes up or down. This is the core metric for a dividend investment strategy.

Formula

\[ Y_{\mathrm{div}} = \frac{D_{\mathrm{annual}}}{P_{\mathrm{current}}} \]
  • \(D_{\mathrm{annual}}\) = total dividends paid per share in the last 12 months (or next 12 months if forecasting)
  • \(P_{\mathrm{current}}\) = current market price per share

Excel → Python

def dividend_yield(annual_dividend_per_share: float, current_price: float) -> float:
    if current_price == 0:
        return 0
    return annual_dividend_per_share / current_price

Concrete examples

Coface (EPA:COFA):

\[ Y_{\mathrm{div}} = \frac{2.15}{16.45} = 0.1307 = \textbf{13.07\%} \]
✓ Excel: Rentabilidade atual estimada = 0.1306990881

NOS (ELI:NOS):

\[ Y_{\mathrm{div}} = \frac{1.40}{5.26} = 0.2662 = \textbf{26.62\%} \]
Caution: A very high dividend yield (like NOS at 26%) can be a warning sign. It sometimes means the market expects the dividend to be cut, or the company is in trouble. The yield is high because the price has fallen, not because dividends increased. Always check the payout ratio and earnings.

8. Required Yield (Rentabilidade Exigida)

What it measures

The minimum dividend yield you accept before buying a stock. This is your personal filter — a strategic rule, not a calculation. Your spreadsheet uses 6.5% for all stocks.

This represents your opportunity cost. If a risk-free government bond pays ~3–4%, you need a meaningful premium to justify the risk of owning individual company stocks. 6.5% is a common threshold in dividend investing communities.

9. Buy Zone Price (Zona de Compra)

What it measures

The maximum price you should pay for a stock to guarantee your required yield. Derived by solving the dividend yield formula for price.

Civil engineering analogy: the maximum bid on a public works tender — you calculate the maximum you can spend and still make the project financially viable.

Formula

Starting from the dividend yield formula and solving for \(P\):

\[ Y = \frac{D}{P} \implies P_{\mathrm{buy}} = \frac{D_{\mathrm{annual}}}{Y_{\mathrm{required}}} \]

Excel → Python

def buy_zone_price(annual_dividend_per_share: float, required_yield: float) -> float:
    if required_yield == 0:
        return 0
    return annual_dividend_per_share / required_yield

Concrete examples

Coface — in the buy zone (current price €16.45):

\[ P_{\mathrm{buy}} = \frac{2.15}{0.065} = \textbf{33.08} \text{ EUR} \quad \Rightarrow \text{ current } 16.45 < 33.08 \text{, IN zone} \]

Amundi — not in the buy zone (current price €87.25):

\[ P_{\mathrm{buy}} = \frac{4.25}{0.065} = \textbf{65.38} \text{ EUR} \quad \Rightarrow \text{ current } 87.25 > 65.38 \text{, NOT in zone} \]
Rental property analogy: If you need 6.5% rental income and the annual rent is €15,600, you should pay no more than €15,600 ÷ 0.065 = €240,000 for the property. Exact same logic.

10. Distance from Buy Zone (Distância da Zona de Compra)

What it measures

How far the current price is from the buy zone limit, as a percentage. Positive = inside the zone (cheap enough). Negative = outside (too expensive).

Formula

\[ D_{\mathrm{zone}} = \frac{P_{\mathrm{buy}} - P_{\mathrm{current}}}{P_{\mathrm{current}}} \]

Excel → Python

def distance_from_buy_zone(buy_zone_price: float, current_price: float) -> float:
    if current_price == 0:
        return 0
    return (buy_zone_price - current_price) / current_price

Concrete examples

Coface (IN zone):

\[ D_{\mathrm{zone}} = \frac{33.08 - 16.45}{16.45} = \frac{16.63}{16.45} = \textbf{+101.1\%} \]
✓ Excel: +1.010755202 (= +101.1%)

Amundi (NOT in zone):

\[ D_{\mathrm{zone}} = \frac{65.38 - 87.25}{87.25} = \frac{-21.87}{87.25} = \textbf{-25.1\%} \]
✓ Excel: -0.2506061274 (= -25.1%)
A distance of +101% (Coface) means the stock would have to double in price before leaving the buy zone. A distance of -25% (Amundi) means the price needs to fall 25% before it becomes attractive.

11. Weighted Portfolio Yield (Rendimento Ponderado)

What it measures

The dividend yield of each stock adjusted by how much of your portfolio it represents. Gives a single percentage for the overall expected dividend income of your entire portfolio.

Formula — per stock

\[ Y_{\mathrm{w},\,i} = Y_{\mathrm{div},\,i} \times W_i \]

Formula — portfolio total

\[ Y_{\mathrm{portfolio}} = \sum_{i=1}^{n} Y_{\mathrm{w},\,i} = \sum_{i=1}^{n} \bigl( Y_{\mathrm{div},\,i} \times W_i \bigr) \]

Excel → Python

def portfolio_yield(holdings: list[dict]) -> float:
    return sum(h["dividend_yield"] * h["weight"] for h in holdings)

Concrete example — SCOR SE (EPA:SCR)

  • Dividend yield: \(2.17 / 32.14 = 6.75\%\)
  • Portfolio weight: \(5.14\%\)
  • Weighted contribution: \(0.0675 \times 0.0514 = 0.347\%\)

Total portfolio yield = 4.30% — for every €100 invested, you expect ~€4.30 in dividends per year.

12. Equal-Weight Yield (Média Igual)

What it measures

What your portfolio yield would be if you invested equally in all stocks. Compared to the actual weighted yield, this reveals whether your allocation is optimized for dividend income.

Formula

\[ Y_{\mathrm{equal}} = \frac{1}{n} \sum_{i=1}^{n} Y_{\mathrm{div},\,i} \]

Excel → Python

def equal_weight_yield(holdings: list[dict]) -> float:
    if not holdings:
        return 0
    return sum(h["dividend_yield"] for h in holdings) / len(holdings)

13. Daily Variation (Variação do Dia)

What it measures

How much a stock moved today compared to yesterday's closing price, in both percentage and absolute euro terms.

Formula — percentage

\[ \Delta_{\%} = \frac{P_{\mathrm{today}} - P_{\mathrm{yesterday}}}{P_{\mathrm{yesterday}}} \]

Formula — absolute (EUR)

\[ \Delta_{\mathrm{EUR}} = P_{\mathrm{today}} - P_{\mathrm{yesterday}} = P_{\mathrm{today}} \times \Delta_{\%} \]

Excel → Python

def daily_variation_percent(today: float, yesterday: float) -> float:
    if yesterday == 0:
        return 0
    return (today - yesterday) / yesterday

def daily_variation_eur(today: float, yesterday: float) -> float:
    return today - yesterday
These values come from a market data API (Yahoo Finance, etc.) — they are not calculated from your transaction history. The app fetches them live.

14. Total Dividends Received (Dividendos Totais)

What it measures

The cumulative cash income received from dividends since the portfolio was created — a simple sum of all dividend transactions.

Formula

\[ D_{\mathrm{total}} = \sum_{i \,\in\, \mathrm{Div}} T_i \]

Where \(T_i\) is the total amount of each dividend transaction.

Excel → Python

def total_dividends(transactions: list[dict]) -> float:
    return sum(
        t["total"]
        for t in transactions
        if t["action"] == "Dividendos"
    )

15. Dividend Contribution (% Contribuição)

What it measures

What percentage of your total dividend income came from each stock. Tells you which stocks are actually working for you as income generators.

Formula

\[ C_i = \frac{D_i}{D_{\mathrm{total}}} \]

Excel → Python

def dividend_contribution(stock_dividends: float, total_dividends: float) -> float:
    if total_dividends == 0:
        return 0
    return stock_dividends / total_dividends

16. Portfolio Cash Position (Cash)

What it measures

The uninvested cash in your brokerage account, derived entirely from transaction history: deposits and dividend payments add cash; purchases subtract it.

Formula

\[ \mathrm{Cash} = \sum_{\mathrm{dep}} T_i + \sum_{\mathrm{sell}} T_i + \sum_{\mathrm{div}} T_i - \sum_{\mathrm{buy}} T_i - \sum_{\mathrm{withdraw}} T_i \]

Excel → Python

def cash_balance(transactions: list[dict]) -> float:
    cash_in  = {"Deposito", "Venda", "Dividendos"}
    cash_out = {"Compra", "Retirada"}
    balance = 0
    for t in transactions:
        if t["action"] in cash_in:
            balance += t["total"]
        elif t["action"] in cash_out:
            balance -= t["total"]
    return balance
